Class: Event

ProAct. Event

new Event(source, target, type)

Constructs a ProAct.Event. The event contains information of the update.

ProAct.Event is part of the core module of ProAct.js.

<static> simple(eventType, subType, value, array) → {ProAct.Event}

Factory method for creating of new ProAct.Events without target and source with ease.

NOTE: For now only array modifying events can be created - remove and splice (you can trigger a value for add).

Using this method we can create for example an event for removing the i-th element from ProAct.Array like this: 
 ProAct.Event.simple('array', 'del', el, array);
This event can be passed to the ProAct.ArrayCore#update method of the core of a ProAct.Array and it will delete the element in it.
Parameters:
Name Type Description
eventType ProAct.Event.Types | String The type of the event. Can be string for ease. For now this method supports only ProAct.Event.Types.array events. It is possible to pass the string 'array' or 'a' for type.
subType ProAct.Array.Operations | String The operation type of the event to create. Can be string or instance of ProAct.Array.Operations. Prossible string values are - 'pop', 'shift', 'deleteElement' or 'del' (at index) and 'splice' for now.
value Object Used a value of the event. For array events this is for example the value to be added or to be removed. It can be index too.
array Array Optional parameter for array events - the array target of the event. It will be set as target. Can be used for determining event's parameters too.
