Class: Val

ProAct. Val

new Val(val, meta)

Constructs a ProAct.Val. The ProAct.Vals are the simplest ProAct.js reactive objects, they have only one property - 'v' and all their methods, extended from ProAct.Actor delegate to it.

Like every object turned to ProAct.js reactive one, the ProAct.Val has a ProAct.ObjectCore managing its single ProAct.Property.

The core can be accessed via: 

   var core = v.p();

ProAct.Val is part of the properties module of ProAct.js.

Parameters:
Name Type Description
val Object The value that will be wrapped and tracked by the ProAct.Val being created.

transform(transformation) → {ProAct.Val}

Adds a new transformation to the list of transformations of the ProAct.Property managing the 'v' field of this.

A transformation is a function or an object that has a call method defined. This function or call method should have one argument and to return a transformed version of it. If the returned value is ProAct.Actor.BadValue, the next transformations are skipped and the updating value/event becomes - bad value.

Every value/event that updates the ProAct.Property managing the 'v' field of this will be transformed using the new transformation.

willUpdate(source, actions, eventData) → {ProAct.Val}

willUpdate() is the method used to notify observers that the ProAct.Property managing the 'v' field of this will be updated.

It uses the ProAct.Actor#defer to defer the listeners of the listening ProAct.Actors. The idea is that everything should be executed in a running ProAct.Flow, so there will be no repetative updates.

The update value will come from the ProAct.Property#makeEvent method and the source parameter will be passed to it.

Parameters:
Name Type Description
source Object The source of the update, for example update of ProAct.Actor, that the ProAct.Property managing the 'v' field of this is observing.

Can be null - no source.

In the most cases ProAct.Event is the source.

actions Array | String A list of actions or a single action to update the listeners that listen to it. If there is no action provided, the actions from ProAct.Actor#defaultActions are used.
eventData Array Data to be passed to the event to be created.
