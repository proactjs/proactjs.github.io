Class: PropertyProvider

ProAct. PropertyProvider

new PropertyProvider()

Constructor for ProAct.PropertyProvider. The ProAct.PropertyProvider is an abstract class.

Many providers can be registered for many kinds of properties.

When a ProAct.js object is initialized its fields are turned into properties. Depending on the type and the name of the field, as well as meta information the valid type of ProAct.Property is created and used. The PropertyProviders have 'filter' method and depending on it the valid kind is decided.

ProAct.PropertyProvider is part of the properties module of ProAct.js.

<static> provide(object, property, meta) → {ProAct.Property}

Provides a ProAct.Property instance using the list of the registered ProAct.PropertyProviders.

The providers are tried in the order of their registration (the order can be changed using ProAct.PropertyProvider.prependProvider).

The ProAct.PropertyProvider#filter method is used to check if a provider is compliant with the passed arguments.

If a compliant provider is found, its ProAct.PropertyProvider#provide method is used to provide the ProAct.Property instance.

Parameters:
Name Type Description
object Object The object to provide a ProAct.Property instance for.
property String The field name of the object to turn into a ProAct.Property.
meta String | Array Meta information to be used for filtering and configuration of the ProAct.Property instance to be provided.

<static> registerProvider(propertyProvider)

Registers a ProAct.PropertyProvider.

The provider is appended in the end of the list of ProAct.PropertyProviders.

When a property must be provided if there is a ProAct.PropertyProvider registered before the passed propertyProvider, with valid filtering for the passed field, it will be used instead.
