    keypropList

    Array static

    Defined in src/js/core/pro.js:555

    Defines a list of the keyword properties that can not be used in ProAct.js objects. The ProAct.Configuration.keyprops option must be set to true in order for this list to be used.

    keyprops

    Boolean static

    Defined in src/js/core/pro.js:541

    If this option is set to true, when a ProAct.js object is created and has properties named as one or more of the properties listed in ProAct.Configuration.keypropList an Error will be thrown.

    In other words declares some of the properties of every ProAct objects as keyword properties.
