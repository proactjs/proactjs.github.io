    ProAct.BufferedStream Class

    Extends ProAct.Stream
    Defined in: src/js/streams/buffered_stream.js:5
    Module: proact-streams

    Constructs a ProAct.BufferedStream. This is a ProAct.Stream with a buffer.

    On new value/event the listeners are not updated, but the value/event is stored in the buffer.

    ProAct.BufferedStream is an abstract class.

    When the buffer is flushed every value/event is emitted to the listeners. In case with property listeners they are updated only once with the last event/value. Good for performance optimizations.

    For example if it is set to stream mouse move events, we don't care for each of the event but for a portion of them.

    ProAct.BufferedStream is part of the proact-streams module of ProAct.js.

    off

    (
    • actions
    • listener
    )
    ProAct.Actor

    Inherited from ProAct.Actor: src/js/core/actor.js:498

    Removes a listener from the passed action.

    If this method is called without parameters, all the listeners for all the actions are removed. The listeners are reset using defaultActions.

    Examples are:

    Removing a listener:

     var listener = function (v) {
   console.log(v);
 };
 actor.on(listener);
 actor.off(listener);

    Or for removing all the listeners attached to an actor:

     actor.off();

    Or for removing all the listeners of a given type attached to an actor:

     actor.off('error');

    Or for removing a listener from different type of actions:

     var listener = function (v) {
   console.log(v);
 };
 actor.on(listener);
 actor.onErr(listener);

 actor.off(['error', 'change'], listener);

    Parameters:

    • actions Array | String

      The action/actions to stop listening for. If this parameter is skipped or null/undefined, the actions from defaultActions are used.

      The actions can be skipped and on their place as first parameter to be passed the listener.

    • listener Object

      The listener to detach. If it is skipped, null or undefined all the listeners are removed from this actor.

    Returns:
