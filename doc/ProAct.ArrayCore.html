Class: ArrayCore

ProAct. ArrayCore

new ArrayCore(array, meta)

Constructs a ProAct.ArrayCore. ProAct.ArrayCore is a ProAct.Core that manages all the updates/listeners for an ProAct.Array.

It is responsible for updating length or index listeners and adding the right ones on read.

ProAct.ArrayCore is part of the arrays module of ProAct.js.

Parameters:
Name Type Description
array Object The shell ProAct.Array arround this core.
meta Object Optional meta data to be used to define the observer-observable behavior of the array.

setup()

Initializes all the index accessors and the length accessor for this's shell array.

For the length on every read, the ProAct.currentCaller is added as a 'length' listener.

For every index on every read, the ProAct.currentCaller is added as an 'index' listener. Listener accessors are defined using ProAct.ArrayCore#defineIndexProp.

ProAct.ArrayCore#addCaller is used to retrieve the current caller and add it as the right listener.

Setting values for an index or the length updates the right listeners.

willUpdate(source, actions, eventData) → {ProAct.Observable}

willUpdate() is the method used to notify observers that this ProAct.Observable will be updated.

It uses the ProAct.Observable#defer to defer the listeners of the listening ProAct.Observables. The idea is that everything should be executed in a running ProAct.Flow, so there will be no repetative updates.

The update value will come from the ProAct.Observable#makeEvent method and the source parameter will be passed to it.

If this ProAct.Observable has a parent ProAct.Observable it will be notified in the running flow as well.

Parameters:
Name Type Description
source Object The source of the update, for example update of ProAct.Observable, that this is observing.

Can be null - no source.

In the most cases ProAct.Event is the source.

actions Array | String A list of actions or a single action to update the listeners that listen to it. If there is no action provided, the actions from ProAct.Observable#defaultActions are used.
eventData Array Data to be passed to the event to be created.
